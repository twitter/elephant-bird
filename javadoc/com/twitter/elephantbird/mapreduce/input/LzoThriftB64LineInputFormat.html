public class LzoThriftB64LineInputFormat<M extends org.apache.thrift.TBase<?,?>>
extends MultiInputFormat<M>

Reads line from an lzo compressed text file, base64 decodes it, and then deserializes that into the Thrift object. Returns pairs.

Do not use LzoThriftB64LineInputFormat.class directly for setting InputFormat class for a job. Use getInputFormatClass() instead.

A small fraction of bad records are tolerated. See LzoRecordReader for more information on error handling.
