public class LzoBinaryB64LineRecordReader<M,W extends BinaryWritable<M>>
extends LzoRecordReader<org.apache.hadoop.io.LongWritable,W>

Reads line from an lzo compressed text file, base64 decodes it, and then deserializes that into the templatized object.

A small fraction of bad records are tolerated. See LzoRecordReader for more information on error handling.
