public abstract class LzoW3CLogRecordReader
extends LzoRecordReader<org.apache.hadoop.io.LongWritable,org.apache.hadoop.io.MapWritable>

A reader for LZO-compressed W3C-style log formatted files. See discussion in the LzoW3CLogInputFormat for more on the format. To use, derive from this class and implement the getFieldDefinitionFile() method.
Most commonly, you can simply call LzoW3CLogInputFormat.newInstance(myFilePath)
