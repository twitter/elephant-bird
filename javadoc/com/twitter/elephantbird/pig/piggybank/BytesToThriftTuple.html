public abstract class BytesToThriftTuple<T extends org.apache.thrift.TBase<?,?>>
extends org.apache.pig.EvalFunc<org.apache.pig.data.Tuple>

This is an abstract UDF for converting serialized Thrift objects into Pig tuples. To create a converter for your Thrift class MyThriftClass, you simply need to extend BytesToThriftTuple with something like this: 

 public class BytesToSimpleLocation extends BytesToThriftTuple<MyThriftClass> {

   public BytesToSimpleLocation() {
     setTypeRef(new TypeRef<MyThriftClass>() {});
   }
 }
