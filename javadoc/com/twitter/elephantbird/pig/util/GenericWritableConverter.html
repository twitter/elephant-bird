public class GenericWritableConverter
extends AbstractWritableConverter<org.apache.hadoop.io.Writable>

Supports conversion between Pig bytearray (DataByteArray) and an arbitrary Writable implementation type. Useful for loading data from a SequenceFile when the key or value must be passed through to output, but otherwise goes untouched by Pig.
