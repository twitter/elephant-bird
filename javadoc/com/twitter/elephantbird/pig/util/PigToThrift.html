public class PigToThrift<T extends org.apache.thrift.TBase<?,?>>
extends java.lang.Object

Converts a Pig Tuple into a Thrift struct. Tuple values should be ordered to match the natural order of Thrift field ordinal values. For example, say we define the following Thrift struct: 

 struct MyThriftType {
   1: i32 f1
   3: i32 f2
   7: i32 f3
 }
Input Tuples are expected to contain field values in order (f1, f2, f3). Tuples may contain fewer values than Thrift struct fields (e.g. only (f1, f2) in the prior example); Any remaining fields will be left unset.
