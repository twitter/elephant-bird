public org.apache.pig.data.Tuple toTuple(com.google.protobuf.Message msg)
Turn a generic message into a Tuple. Individual fields that are enums are converted into their string equivalents. Fields that are not filled out in the protobuf are set to null, unless there is a default field value in which case that is used instead.

getFieldValue

protected java.lang.Object getFieldValue(com.google.protobuf.Message msg,
                                         com.google.protobuf.Descriptors.FieldDescriptor fieldDescriptor)
A utility function for getting the value of a field in a protobuf message. It first tries the literal set value in the protobuf's field list. If the value isn't set, and the field has a default value, it uses that. Otherwise, it returns null.

Parameters:
msg - the protobuf message
fieldDescriptor - the descriptor object for the given field.
Returns:
the value of the field, or null if none can be assigned.
