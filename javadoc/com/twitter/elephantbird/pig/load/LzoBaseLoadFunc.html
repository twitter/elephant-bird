public abstract class LzoBaseLoadFunc
extends org.apache.pig.LoadFunc
implements org.apache.pig.LoadMetadata, org.apache.pig.LoadPushDown

This class handles LZO-decoding and slicing input LZO files. It expects the filenames to end in .lzo, otherwise it assumes they are not compressed and skips them. TODO: Improve the logic to accept a mixture of lzo and non-lzo files.
