public class LzoRegexLoader
extends LzoBaseRegexLoader

LzoRegexLoader extends LzoBaseRegexLoader, allowing regular expressions to be passed by argument through pig latin via a line like: A = LOAD 'test.txt' USING com.twitter.elephantbird.pig.storage.LzoRegexLoader('(\\d+)::(\\w+)\|\|(\\w+)'); which would parse lines like 1::one||i 2::two||ii 3::three--iii into arrays like {1, "one", "i"}, {2, "two", "ii"}, {3, "three", "iii"}
